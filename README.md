## Project Overview

This repository contains my Digital Literacy evaluated course project. The project demonstrates different aspects of digital literacy including infographic creation, building a professional digital portfolio, exploring coding platforms, learning professional email etiquette, and understanding cybercrime awareness.

---

## Tasks Included

**Task 1 – Digital Literacy Awareness Infographic**  
Created an infographic using Canva explaining digital literacy, tools, safe internet practices, and professional communication.

**Task 2 – Student Digital Portfolio**  
Created and updated profiles on GitHub, LinkedIn, and Kaggle to build a professional digital presence.

**Task 3 – Coding Platform & Google Form**  
Solved beginner-level coding problems on CodeChef and created a Digital Literacy Awareness Quiz using Google Forms.

**Task 4 – Professional Email & Etiquette Guide**  
Wrote professional emails and created a social media do’s and don’ts checklist.

**Task 5 – Cybercrime Awareness Case Study**  
Prepared a case study on phishing and created a prevention checklist for online safety.
